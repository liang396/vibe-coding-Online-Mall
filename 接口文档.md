# 在线买卖商城接口文档

## 1. 用户接口

### 1.1 用户注册

* **URL**: `/api/users/register`
* **方法**: POST
* **参数**:

  | 参数       | 类型     | 必填 | 描述              |
  | -------- | ------ | -- | --------------- |
  | username | string | 是  | 用户名             |
  | email    | string | 是  | 邮箱              |
  | password | string | 是  | 密码              |
  | role     | string | 否  | 角色：buyer/seller |
* **返回**:

  ```json
  {"success": true, "user_id": 1}
  ```

### 1.2 用户登录

* **URL**: `/api/users/login`
* **方法**: POST
* **参数**:

  | 参数       | 类型     | 必填 | 描述  |
  | -------- | ------ | -- | --- |
  | username | string | 是  | 用户名 |
  | password | string | 是  | 密码  |
* **返回**:

  ```json
  {"success": true, "token": "JWT_TOKEN"}
  ```

### 1.3 获取用户信息

* **URL**: `/api/users/{user_id}`
* **方法**: GET
* **返回**:

  ```json
  {"user_id":1, "username":"张三", "email":"test@example.com", "role":"buyer"}
  ```

### 1.4 更新用户信息

* **URL**: `/api/users/{user_id}`
* **方法**: PUT
* **参数**:

  | 参数       | 类型     | 必填 | 描述  |
  | -------- | ------ | -- | --- |
  | username | string | 否  | 用户名 |
  | email    | string | 否  | 邮箱  |
  | phone    | string | 否  | 电话  |
* **返回**:

  ```json
  {"success": true}
  ```

## 2. 商品接口

### 2.1 获取商品列表

* **URL**: `/api/products`
* **方法**: GET
* **参数**:

  | 参数          | 类型     | 必填 | 描述    |
  | ----------- | ------ | -- | ----- |
  | category_id | int    | 否  | 分类ID  |
  | keyword     | string | 否  | 搜索关键词 |
  | page        | int    | 否  | 页码    |
  | size        | int    | 否  | 每页数量  |
* **返回**:

  ```json
  [{"product_id":1,"name":"商品A","price":100.00}]
  ```

### 2.2 获取商品详情

* **URL**: `/api/products/{product_id}`
* **方法**: GET
* **返回**:

  ```json
  {"product_id":1,"name":"商品A","description":"详细描述","price":100.00,"stock":10}
  ```

### 2.3 发布商品

* **URL**: `/api/products`
* **方法**: POST
* **参数**:

  | 参数          | 类型      | 必填 | 描述   |
  | ----------- | ------- | -- | ---- |
  | name        | string  | 是  | 商品名称 |
  | description | string  | 否  | 商品描述 |
  | price       | decimal | 是  | 价格   |
  | stock       | int     | 是  | 库存   |
  | category_id | int     | 是  | 分类ID |
* **返回**:

  ```json
  {"success": true, "product_id": 1}
  ```

### 2.4 更新商品

* **URL**: `/api/products/{product_id}`
* **方法**: PUT
* **参数**: 同发布商品，可选更新字段
* **返回**:

  ```json
  {"success": true}
  ```

### 2.5 删除商品

* **URL**: `/api/products/{product_id}`
* **方法**: DELETE
* **返回**:

  ```json
  {"success": true}
  ```

## 3. 订单接口

### 3.1 创建订单

* **URL**: `/api/orders`
* **方法**: POST
* **参数**:

  | 参数       | 类型    | 必填 | 描述                            |
  | -------- | ----- | -- | ----------------------------- |
  | buyer_id | int   | 是  | 买家ID                          |
  | items    | array | 是  | 商品列表 [{product_id, quantity}] |
* **返回**:

  ```json
  {"success": true, "order_id": 1}
  ```

### 3.2 获取订单列表

* **URL**: `/api/orders`
* **方法**: GET
* **参数**:

  | 参数      | 类型     | 必填 | 描述   |
  | ------- | ------ | -- | ---- |
  | user_id | int    | 否  | 用户ID |
  | status  | string | 否  | 订单状态 |
* **返回**:

  ```json
  [{"order_id":1,"total_price":200.00,"status":"pending"}]
  ```

### 3.3 获取订单详情

* **URL**: `/api/orders/{order_id}`
* **方法**: GET
* **返回**:

  ```json
  {"order_id":1,"buyer_id":1,"items":[{"product_id":1,"quantity":2}],"status":"pending"}
  ```

### 3.4 更新订单状态

* **URL**: `/api/orders/{order_id}/status`
* **方法**: PUT
* **参数**:

  | 参数     | 类型     | 必填 | 描述                                       |
  | ------ | ------ | -- | ---------------------------------------- |
  | status | string | 是  | pending/paid/shipped/completed/cancelled |
* **返回**:

  ```json
  {"success": true}
  ```

## 4. 评论接口

### 4.1 添加评论

* **URL**: `/api/reviews`
* **方法**: POST
* **参数**:

  | 参数         | 类型     | 必填 | 描述    |
  | ---------- | ------ | -- | ----- |
  | product_id | int    | 是  | 商品ID  |
  | user_id    | int    | 是  | 用户ID  |
  | rating     | int    | 是  | 评分1-5 |
  | comment    | string | 否  | 评论内容  |
* **返回**:

  ```json
  {"success": true, "review_id": 1}
  ```

### 4.2 获取商品评论

* **URL**: `/api/products/{product_id}/reviews`
* **方法**: GET
* **返回**:

  ```json
  [{"review_id":1,"user_id":2,"rating":5,"comment":"很好"}]
  ```
